# ファイルの説明

## PSM.html

CSVファイルを入力すると PSM.jsによってファイルの解析が行われ、出力結果とグラフを表示します。

## PSM.js

CSVファイルを解析するプログラムが書かれています。

## 出力結果の表示手順

1. PSM.htmlをブラウザを使って表示する

2. 表示されたPSM.html上からCSVファイルをアップロードする

3. アップロードを行うと、最高価格、妥協価格、理想価格、最低品質保証価格の表示およびPSM分析のグラフの表示が行われる
